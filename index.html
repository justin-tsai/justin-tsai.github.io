Projects

Spellbreaker VR

Spellbreaker VR is a game being currently made in my spare time. Set in a cosmic space-like environment, the player uses swords to slash through a flurry of magic projectiles aimed at them. This is my take on Beat Saber, but with a small twist of being in a different environment and added mechanics. I'm currently gaining a lot of experience with creating particle systems and using lerping. I'm excited to see what else I can learn and apply to this project.

Made in Unity using C#.

Isolation

Isolation is a game that can be played by running the program on the console. The goal of the game is to trap the other player by using previous moves to block them off. The player plays against an computer that makes its moves using a min-max algorithm with alpha-beta pruning and iterative deepening. This was my first time implementing an algorithm for an AI, and I was pretty happy with the results.

Made in Eclipse using Java.

Below Haven

Below Haven is a project I developed in a team of 5 that made for my game development class. The idea for the game was heavily inspired by the series Made In Abyss. This is a 2D game in which the player has to travel underground to discover treasure and riches, but the levels and enemies get harder as they venture further below. I was mainly in charge of finding assets, coding the scripts for movement and enemy interactions, and all of the audio. We were all beginners to Unity so it was a great learning experience.

Made in Unity using C#.

VR Combination and Permutation

VR Combination and Permutation is a research project that aims to test the player's understanding of combination and permutation by having them solve puzzles using each concept to escape a trapped room. Statistics are given for each puzzle to display the number of possible guesses to the solution. This project was presented to CPP's 13th Annual Science Research Symposium. I am also proud to say that this was the first project I made with Unity.

Made in Unity using C#.
