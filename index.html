Projects

Spellbreaker VR

Spellbreaker VR is a game made in Unity. Set in a cosmic space-like environment, the player uses swords to slash through a flurry of magic projectiles aimed at them. This is my take on Beat Saber, but with a small twist of being in a different environment and added mechanics. I was responsible for game design, mechanics, audio, particle systems, and player controls.

Made in Unity using C#.

VR Combination and Permutation

VR Combination and Permutation is a research project that aims to test the player's understanding of combination and permutation by having them solve puzzles using each concept to escape a trapped room. Each puzzle has statistics to display the number of possible guesses to the solution. The project was presented to Cal Poly Pomona's 13th Annual Science Research Symposium. I was responsible for environment interactions, audio, and player controls.

Made in Unity using C#.

Isolation

Isolation is a game that can be played by running the program on the console. The goal of the game is to trap the other player by using previous moves to block them off. The player plays against an computer that makes its moves using a min-max algorithm with alpha-beta pruning and iterative deepening.

Made in Eclipse using Java.
